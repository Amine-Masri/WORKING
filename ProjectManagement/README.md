# Spring CRUD Example - Simple Project Management App
This is a simple project management application that I built to learn Spring Boot.
## Used technologies:
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- MySQL
- Thymeleaf
- Vue.js 2.0 & Axios
- Bootstrap
- HTML / CSS
